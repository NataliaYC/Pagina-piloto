Descripción del Proyecto
Este repositorio contiene el código fuente para el sitio web estático de Tienda Mira Que Chuli, una tienda dedicada a la venta de productos 100% originales y licenciados de diversas marcas populares.

El sitio web está diseñado como parte de un ejercicio práctico para una actividad de curso se sense desarrollador full stack java trainee.
como objeto de aprendizaje para subir con git commits ael repositoprio creado en GitHub.
La pagina sirve para poder a futuro crear un portafolio y terminar en un e-comerse para la tienda que es real.
mientras tanto su utilidad es para dar a conocer eventos que se acercan en los meses venideros como medio de difusión.
ecnologías Utilizadas
El proyecto fue desarrollado utilizando tecnologías web estándar y el popular framework front-end:

HTML5: Estructura y contenido del sitio.

CSS3: Estilos personalizados (asset/css/style.css).

Bootstrap 5.3: Framework CSS para el diseño responsive, el sistema de navegación (Navbar) y el layout general.

JavaScript: Incluido mediante bundle de Bootstrap para funcionalidades como el dropdown en la barra de navegación.

nstalación y Visualización
Dado que este es un proyecto web estático (solo HTML/CSS/JS), la instalación es muy sencilla.

1. Clonar el Repositorio
Clona este repositorio en tu máquina local:
git clone https://aws.amazon.com/es/what-is/repo/
https://github.com/NataliaYC/Pagina-piloto.git

2. Estructura de Archivos
Asegúrate de que la estructura de archivos sea la siguiente para que las referencias a CSS e imágenes funcionen correctamente:

/
├── index.html
├── asset/
│   ├── css/
│   │   └── style.css
│   └── img/
│       ├── logo_tienda_p.jpg
│       └── instagram.svg
└── README.md

3. Abrir la Página
Simplemente abre el archivo index.html en tu navegador web de preferencia (Chrome, Firefox, Edge, etc.). No se requiere un servidor web para la visualización básica.
Características Principales
Barra de Navegación Responsive: Implementada con Bootstrap, incluyendo un menú desplegable para las categorías de productos (Disney, Marvel, Otros).

Información de Eventos: Sección destacada para el anuncio de la nueva línea Disney Navidad y la participación en la Feria Navideña de la Villa México.

Listado de Productos: Mención de los tipos de productos más populares (Tazones Disney, Mugs Marvel, Botellas Star Wars).

Integración Social: Enlace directo al perfil de Instagram de la tienda.
